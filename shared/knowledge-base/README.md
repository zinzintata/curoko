# ナレッジベース

各部門が蓄積した汎用知識をここに置きます。特定プロジェクトに紐づかない、再利用可能な知見・チェックリスト・手順書など。

## カテゴリ例

- `brand/` — ブランドガイドライン、トーン&マナー
- `clients/` — クライアント情報(個人情報に注意)
- `tech-stack/` — 使用技術の選定理由、設定メモ
- `templates/` — 汎用的な文書テンプレ
- `playbooks/` — 繰り返し発生する業務の手順書

## 命名規則

- ファイル名: `kebab-case.md`
- 日付が関係するもの: `YYYY-MM-DD-<topic>.md`
